Bienvenido al repositorio oficial de CASA NOVA, una plataforma web diseñada para la comercialización de propiedades de alta gama. Este sitio web se destaca por su diseño minimalista, interactividad y enfoque en la experiencia del usuario.

🚀 Características principales
 Interfaz Minimalista: Diseño limpio con tipografía impactante.
 Galería Interactiva: Sistema de modales para visualización de propiedades con detalle fotográfico.
 Sección 50/50: Layout responsivo para destacar propiedades exclusivas.
 Responsive Design: Adaptable a dispositivos móviles y escritorio.
